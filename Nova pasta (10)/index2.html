CookieScript Wiki

Introducao

CookieScript e uma linguagem de script hibrida que combina operacoes de sistema (Python) com operacoes web (JavaScript). Desenvolvida para facilitar a criacao de ferramentas que precisam interagir tanto com o sistema operacional quanto com servicos web.

O projeto permite escrever scripts simples que podem executar tarefas como manipulacao de arquivos, requisicoes HTTP, criptografia, operacoes matematicas, manipulacao de strings, acesso a bancos de dados e muito mais, tudo em uma sintaxe unificada.

Atualmente conta com 19 modulos integrados.

Instalacao

Download

Baixe o projeto completo em ZIP atraves deste link: Download CookieScript (Google Drive)

Pre-requisitos

  • Python 3.8+
  • Node.js 14+

Passos de Instalacao

  1. Extraia o arquivo ZIP
  2. Abra o terminal na pasta extraida
  3. Instale dependencias Python:
    pip install -r requirements.txt
  4. Instale dependencias Node.js:
    npm install

Como Usar

Executando Scripts

python run_cookiescript.py meu_script.cookie

Ou interativamente:

python run_cookiescript.py

Testando WebSocket

python server.py
# Em outro terminal:
node cookiescript.js

CookieScript IDE

python cookie_ide_app.py

Abra: http://127.0.0.1:9000/

Linguagem CookieScript

Sintaxe Basica

Tipos: int, float, str, bool, list, dict.

numero = 42 + 8
texto = "Ola" + " Mundo"

Estruturas de Controle

if numero > 5 { status = "maior" } else { status = "menor" }
for item in lista { soma = math.somar(soma, item) }
while contador < 10 { contador = math.somar(contador, 1) }

Tratamento de Erros

try { resultado = math.dividir(10, 0) } catch { erro = "Divisao por zero: " + error_message }

FAQ

Depende do uso. Use com responsabilidade.

Posso adicionar novos modulos?
