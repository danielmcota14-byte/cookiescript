CookieScript Wiki

Introdução

CookieScript é uma plataforma de desenvolvimento com IA potencializada que combina operacoes de sistema com IA avancada. Permite escrever scripts simples que interagem com o sistema operacional e servicos web, tudo com assistencia de IA.

Recursos principais:

  • editor de codigo profissional
  • IA local (DeepSeek/Phi-2)
  • integração Groq API (Llama, Mixtral)
  • fluxo de IAs potencializado (Local + Groq)
  • gerador e pesquisador de codigo com IA

Instalação

Pre-requisitos

  • Python 3.8+
  • Node.js 14+ (opcional)

Passos

  1. Execute o arquivo CookieScript.py para instalar automaticamente todas as dependencias
  2. Aguarde o download dos modelos de IA
  3. Acesse http://localhost:8080 no navegador
python CookieScript.py

Como Usar

CookieScript IDE

python cookie_ide_app.py

Acesse: http://127.0.0.1:8080

Funcionalidades

  • Editor: Abra, edite e salve arquivos .cookiescript
  • Executar: Execute codigo Python, JavaScript ou CookieScript
  • IA Assistant: Pergunte para a IA gerar ou refinar codigo
  • Gerador: Descreva o que voce quer e a IA gera o codigo

IA Potencializada

Fluxo entre IAs

A CookieScript oferece um fluxo unico de potencialização de IAs:

  1. Sua pergunta é enviada para a IA Local (DeepSeek/Phi-2)
  2. Groq API gera uma resposta baseada no contexto
  3. IA Local refina e melhora a resposta do Groq

Como usar

  1. Acesse index1.html (AI Assistant)
  2. Selecione "Cookie Potencializada" no seletor
  3. Digite sua pergunta e envie

FAQ

O que diferencia CookieScript?

A integracao entre sistema operacional, web services e IA potencializada em um unico fluxo.

A IA Local precisa de GPU?

Não. Funciona via CPU, mas pode usar GPU se disponivel.

Como funciona o fluxo de IAs?

O Groq responde rapidamente, depois a IA Local refina a resposta para maior qualidade.
